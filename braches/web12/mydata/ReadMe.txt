Welcome to the ASP.NET AJAX Control Toolkit!


Included in this package:
* SampleWebsite - A web site that demonstrates how to use the Toolkit controls
  and contains the Toolkit documentation
* AjaxControlExtender - A VSI installer that will install templates onto your
  machine to enable creation of new ASP.NET AJAX extender controls using Visual
  Studio 2005

To get started, simply open the SampleWebSite in Visual Studio 2005, and run
it.  This will allow you to experiment with the different controls.

The web site contains walkthroughs covering:
* Setting up your machine to use ASP.NET AJAX Control Extenders
* Using control extenders with your ASP.NET web site
* Creating new control extenders

Please open the sample web site or http://ajax.asp.net/ajaxtoolkit/ for more
details!
